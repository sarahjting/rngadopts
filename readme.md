# rngadopt

FlightRising-style colorwheel RNG pet generator

# Setup
1. Set up your `.env` --
  ```
  cp .env.example .env
  ```
2. Run the Docker container --
  ```
  docker compose up -d
  ```
3. Go to https://localhost:8000 -- this is now your admin panel. You can use it to upload your pet assets.
